ABOUT UTOPIC SENSE:

It is a project that started as an academic project in 2019. It was implemented during that time so the application was tested 
from version 2018.3.0f2 to versions above.

HOW TO USE:

After donwloading the package be sure that the Resources and ScriptsUS folders are
inside your Assets folder package otherwise the application won't work properly. 

Copy the folders mentioned before into your
Assets folder in your project if they are not in the correct place.

Window >UtopicSense to access the interface of the application.
